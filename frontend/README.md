# Aimers Frontend

Next.js App Router + TypeScript + Tailwind CSS for the Aimers platform.

## Scripts

```bash
npm run dev      # http://localhost:3000
npm run build
npm run start
npm run lint
```

## Stack

- TanStack Query
- Recharts
- Outfit + Space Grotesk fonts
- Brand logo: `/public/aimers-logo.png`
